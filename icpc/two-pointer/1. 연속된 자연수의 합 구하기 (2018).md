# 아이디어
- 처음에는 합이 될때까지 (`sum < n`) 모두 더해나간다. 그리고 합을 기록한다. 
- 합에 도달했을 경우 (`sum == n`) `right+=1` 을 통해 윈도우의 사이즈를 오른쪽으로 계속넓힌다. 그리고 현재까지의 합을 기록한다.
- 합을 넘어설 경우 (`sum > n`) `left+=1` 을 통해 윈도우의 사이즈를 왼쪽 부분을 한칸 줄인다. 그리고 이전까지의 합에서 left 를 뺀 값으로 현재까지의 합을 업데이트한다.
<br>

# 그림
책의 그림을 보면 확실하게 이해된다. 저작권 문제로 인해 그림을 캡처하진 않았다.<br/>

# python
```python
import sys

N: int = int(sys.stdin.readline().strip())

left: int = 0
right: int = 0
total = 0
cnt = 0

while right <= N:
    if total == N:
        cnt+=1
        right+=1
        total+=right
    elif total > N:
        left+=1
        total -= left
    elif total < N:
        right+=1
        total += right

print(f"{cnt}")
```
