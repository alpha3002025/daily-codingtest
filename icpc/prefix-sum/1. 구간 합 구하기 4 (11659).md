조금 지저분하게 풀었다. 인덱스에 -1 을 하는 것을 나중에 알았다. start, end 인덱스가 헷갈리기도 했었다. 사실 이런 문제는 굉장히 깔끔하게 빨리 풀수 있어야 한다.
```python
import sys

N, M = map(int, sys.stdin.readline().strip().split())

numbers = list(map(int, sys.stdin.readline().strip().split()))

queries = [list(map(int, sys.stdin.readline().strip().split())) for _ in range(M)]


section = [0] * len(numbers)
section[0] = numbers[0]

for i in range(1, len(numbers)):
    section[i] = section[i-1] + numbers[i]

for q in queries:
    start, end = q[0]-1, q[1]-1
    if start == 0:
        print(f"{section[end]}")
    elif start == end:
        curr = section[end] - section[end-1]
        print(f"{curr}")
    else:
        curr = section[end]-section[start-1]
        print(f"{curr}")
```

좋은 풀이를 보는 과정은 좋은 풀이를 보고 실전에서 한정된 환경에서 더 좋은 접근방식으로 나를 좀더 쉽게 움직여갈수 있게 연결하기 위한 과정이다.
```python
import sys
input = sys.stdin.readline
suNo,quizNo = map(int, input().split())

prifix_sum = [0]

for i in numbers:
    temp = temp + i
    prifix_sum.append(temp) #합 배열 만들기
for i in range(quizNo):
    s, e = map(int, input().split())
    print(prifix_sum[e]-prifix_sum[s-1]) #합 배열에서 구간합 구하기
```
