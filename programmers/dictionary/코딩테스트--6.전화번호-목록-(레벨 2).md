나는 아래 풀이 같은 방식을 '그리디' 라는 유형으로 기억을 자주했는데, 프로그래머스 문제의 경우 어떤 문제를 보고 '그리디!'이렇게 접근하는 방식이 그렇게 딱 맞지는 않았다. 

아무튼 이번 문제는 `정렬 후 검색` 의 유형인데, 다른 문제에서도 자주 보이는 유형이다.

```python
def solution(phone_book):
    phone_book.sort()
    
    for i in range(len(phone_book)-1):
        if phone_book[i+1].startswith(phone_book[i]):
            return False
    return True
```
