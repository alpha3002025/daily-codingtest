- answer = (종류 1 의 의상 수 + 1) x (종류 2의 의상수 +1) ... x (종류 n 의 의상수 + 1) - 1
- +1을 하는 것은 해당 종류의 의상을 안입는 경우를 의미
- -1 을 하는 것은 발가벗게되는 케이스를 제외하는 것을 의미 (홀딱 벗으면 안돼)

```python
from collections import defaultdict
def solution(clothes):
    PRODUCT, KIND = (0,1)
    table = defaultdict(list)
    
    for c in clothes:
        table[c[KIND]].append(c[PRODUCT])
    
    answer = 1
    for kind, value in table.items():
        answer *= (len(table[kind])+1)
        
    return answer-1
```
