# 점 찍기

## 문제 설명
좌표평면 위 원점 `(0, 0)`으로부터 거리가 `d` 이하인 위치 중에서,
x좌표와 y좌표가 모두 `k`의 배수인 점의 개수를 구하는 문제입니다.

## 풀이 개념
**수학 (방정식)**을 이용해 반복문의 횟수를 줄이는 것이 핵심입니다.
단순 이중 반복문은 `d`가 최대 1,000,000이므로 `O((d/k)^2)`가 되어 시간 초과가 날 수 있습니다.
`x^2 + y^2 <= d^2` 부등식을 이용해, `x`가 정해졌을 때 가능한 `y`의 최대값을 구하면 `O(d/k)`로 해결 가능합니다.

1. `x`를 0부터 `d`까지 `k`씩 증가시킵니다.
2. 각 `x`에 대해, `y^2 <= d^2 - x^2` 이므로 `max_y = floor(sqrt(d^2 - x^2))` 입니다.
3. 0부터 `max_y`까지 `k`의 배수의 개수는 `max_y // k + 1` 개입니다. (0 포함)
4. 이를 모두 더합니다.

## 코드 (Python)

```python
import math

def solution(k, d):
    answer = 0
    
    # x 좌표를 0, k, 2k, ... 순으로 증가
    for x in range(0, d + 1, k):
        # 피타고라스 정리로 가능한 최대 y 거리 계산
        max_dist_y = math.isqrt(d**2 - x**2)
        
        # 0부터 max_dist_y 사이의 k 배수 개수
        count_y = max_dist_y // k + 1
        answer += count_y
        
    return answer
```
