# 택배 상자 꺼내기 (PCCP 기출 1번)

## 문제 설명
1번부터 n번까지 번호가 매겨진 택배 상자를 창고에 보관합니다. 상자는 다음과 같은 규칙으로 쌓습니다.
1. 가로로 `w`개씩 상자를 놓습니다.
2. 1번 상자부터 순서대로 놓되,
   - 1번째 층(0행)은 왼쪽에서 오른쪽으로
   - 2번째 층(1행)은 오른쪽에서 왼쪽으로
   - 3번째 층(2행)은 다시 왼쪽에서 오른쪽으로
   - 지그재그(?!) 방식으로 계속 쌓아 올립니다.

특정 상자 `num`을 꺼내려 할 때, 그 상자 위에 있는 모든 상자를 먼저 꺼내야 합니다.
`num`번 상자를 꺼내기 위해 총 몇 개의 상자를 꺼내야 하는지 구하는 문제입니다 (본인 포함).

## 제한 사항
- $2 \le n \le 100$
- $1 \le w \le 10$
- $1 \le num \le n$

## 풀이 접근
상자들이 격자 형태로 쌓여 있다고 볼 수 있습니다. 격자의 행(Row)은 0부터 시작하여 증가하고, 열(Col)은 0부터 `w-1`까지입니다.

1. **타겟 상자의 위치 찾기**:
   - `num` 상자가 몇 번째 층(Row)에 있는지 구합니다. `(num - 1) // w`
   - 해당 층에서 몇 번째 열(Col)에 있는지 구합니다.
     - 짝수 층(0, 2, ...): 왼쪽 $\to$ 오른쪽 순서이므로 `(num - 1) % w`가 열 번호입니다.
     - 홀수 층(1, 3, ...): 오른쪽 $\to$ 왼쪽 순서이므로 `w - 1 - ((num - 1) % w)`가 열 번호입니다.

2. **위에 있는 상자 카운트**:
   - 타겟 상자의 `Col`은 고정입니다.
   - 타겟 상자의 `Row`부터 층을 하나씩 높여가며(`Row + 1`, `Row + 2`, ...) 해당 `Col`에 상자가 존재하는지 확인합니다.
   - 각 층의 해당 `Col`에 놓일 상자의 번호를 계산하고, 그 번호가 `n` 이하이면 상자가 있는 것으로 간주하여 카운트합니다.
   - 상자 번호가 `n`을 초과하면 더 이상 위에 상자가 없는 것이므로 종료합니다.

## 코드 (Python)

```python
def solution(n, w, num):
    # 1. 찾고자 하는 상자(num)의 위치(행, 열) 계산
    target_row = (num - 1) // w
    target_col_offset = (num - 1) % w
    
    # 짝수번째 층: 왼쪽 -> 오른쪽, 홀수번째 층: 오른쪽 -> 왼쪽
    if target_row % 2 == 0:
        target_col = target_col_offset
    else:
        target_col = w - 1 - target_col_offset
        
    count = 0
    curr_row = target_row
    
    # 2. 해당 위치(target_col)의 위에 있는 상자들을 확인하며 카운트
    while True:
        # 현재 층의 시작 번호 계산 (0층:1, 1층:w+1, ...)
        start_num = curr_row * w + 1
        
        # 현재 층의 해당 열(target_col)에 있는 상자 번호 계산
        if curr_row % 2 == 0:
            curr_num = start_num + target_col
        else:
            curr_num = start_num + (w - 1 - target_col)
            
        # 상자가 존재하면 카운트
        if curr_num <= n:
            count += 1
            curr_row += 1
        else:
            break
            
    return count
```
