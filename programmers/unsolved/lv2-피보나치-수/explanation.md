# 피보나치 수

## 문제 설명
피보나치 수 $F(n) = F(n-1) + F(n-2)$를 구하여 1234567로 나눈 나머지를 반환하세요. ($n \ge 2$)

### 핵심 개념
1.  **동적 계획법 (DP) / 반복문**: 재귀(오버헤드 큼)보다는 반복문이나 메모이제이션을 사용해야 합니다.
2.  **모듈러 연산 성질**: $(A + B) \% M = ((A \% M) + (B \% M)) \% M$.
    - 매번 더할 때마다 1234567로 나눠도 결과값은 같습니다. 숫자가 너무 커지는 것을 방지할 수 있습니다.

## Python 풀이

```python
def solution(n):
    # F(0)=0, F(1)=1
    a, b = 0, 1
    
    for _ in range(2, n + 1):
        a, b = b, (a + b) % 1234567
        
    return b
```

### 코드 설명
- 리스트를 만들어서 `dp[i]`를 저장해도 되지만, 직전 두 값(`a`, `b`)만 필요하므로 변수 2개만 써서 메모리를 아낄 수 있습니다. ($O(N)$ 시간, $O(1)$ 공간)
- `b`가 최종 $F(n)$이 됩니다.
