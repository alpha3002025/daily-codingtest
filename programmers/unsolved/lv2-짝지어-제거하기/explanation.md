# 짝지어 제거하기

## 문제 설명
문자열에서 같은 알파벳이 2개 붙어있는 짝을 찾아 제거합니다.
제거 후 앞뒤로 이어붙이고, 다시 짝을 찾아 제거하는 과정을 반복합니다.
문자열이 모두 제거되면 1, 아니면 0을 반환합니다.
예: `baabaa` -> `b aa baa` -> `bbaa` -> `bb aa` -> `aa` -> `` (성공)

### 핵심 개념
1.  **스택 (Stack)**: 문자열을 앞에서부터 하나씩 확인하며, 스택의 `top`과 같으면 짝지어 제거(`pop`)하고, 다르면 넣습니다(`push`).
    - 이 방식은 괄호 짝 맞추기와 동일한 원리입니다.
    - $O(N)$으로 해결 가능합니다. (문자열 재조합 방식은 $O(N^2)$이라 시간 초과 날 수 있음)

## Python 풀이

```python
def solution(s):
    stack = []
    
    for char in s:
        # 스택이 비어있지 않고, 마지막 문자가 현재 문자와 같으면 제거
        if stack and stack[-1] == char:
            stack.pop()
        else:
            stack.append(char)
            
    # 스택이 비어있으면 모두 제거된 것
    if not stack:
        return 1
    else:
        return 0
```

### 코드 설명
- `stack`을 이용하면 "제거 후 이어붙이는" 연산을 실제로 수행할 필요가 없습니다. 스택 자체가 제거되지 않은 앞부분 문자들을 유지하고 있기 때문입니다.
- $N$이 최대 1,000,000이므로 반드시 $O(N)$ 알고리즘을 써야 합니다.
