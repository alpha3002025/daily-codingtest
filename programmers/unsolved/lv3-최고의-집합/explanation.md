# 최고의 집합

## 문제 설명
자연수 `n`개로 이루어진 집합 중, 합이 `s`가 되면서 각 원소의 곱이 최대가 되는 집합을 구하세요.
오름차순으로 정렬하여 반환합니다. 불가능하면 `[-1]`.

## 문제 해결 전략

**수학 / 그리디**.
- 합이 일정할 때 곱이 최대가 되려면 **숫자들이 최대한 균등하게 분포**되어야 합니다 (차이가 작아야 함).
  - 예: 합 10 -> 1+9=9, 5+5=25.
- 따라서 `s`를 `n`으로 나눈 몫(`quotient`)을 기본으로 하고, 나머지(`remainder`)만큼을 1씩 더해줍니다.
- 나머지는 뒤쪽(큰 수)부터 분배하면 오름차순 정렬 상태를 유지하기 좋습니다.

## Python 코드

```python
def solution(n, s):
    if n > s:
        return [-1]
        
    q = s // n
    r = s % n
    
    # 기본값 q로 채움
    answer = [q] * n
    
    # 나머지 r개를 뒤에서부터 1씩 더해줌 (오름차순 유지를 위해)
    # 뒤쪽 원소들이 커야 곱이 최대? -> 사실 값들이 가까워야 함.
    # [3, 3, 3]에서 나머지 2 -> [3, 4, 4]가 [3, 3, 5]보다 곱이 큼 (48 vs 45).
    # 따라서 1씩 골고루 더해주는 게 맞음.
    
    for i in range(r):
        answer[n - 1 - i] += 1
        
    return answer
```
