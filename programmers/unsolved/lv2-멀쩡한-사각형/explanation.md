# 멀쩡한 사각형

## link
- https://school.programmers.co.kr/learn/courses/30/lessons/62048

## 문제 설명
가로 길이가 $W$, 세로 길이가 $H$인 직사각형 종이에 대각선을 그었을 때, 대각선이 지나가는 1x1 크기의 단위 정사각형들의 개수를 구하여, 전체 사각형 개수에서 뺀 값을(사용 가능한 정사각형 개수) 반환하는 문제입니다.

### 핵심 개념
1.  **패턴 찾기 / 최대공약수(GCD)**:
    - 작은 예시($W=2, H=3$)를 그려보면, 대각선이 지나는 칸은 4개입니다.
    - 대각선이 지나는 정사각형의 개수 공식: $W + H - \gcd(W, H)$
    - 여기서 $\gcd(W, H)$는 $W$와 $H$의 최대공약수입니다.
2.  **공식 유도**:
    - 직사각형은 $\gcd(W, H)$만큼의 작은 반복되는 패턴(블록)으로 나뉩니다.
    - 작은 블록의 가로 $w = W / gcd$, 세로 $h = H / gcd$.
    - 이 작은 블록($w \times h$)에서 대각선이 지나는 칸 수는 $w + h - 1$개입니다.
    - 전체 개수는 $(w + h - 1) \times \gcd(W, H) = W + H - \gcd(W, H)$ 가 됩니다.

## Python 풀이

```python
import math

def solution(w, h):
    # 전체 칸 수 - 사용할 수 없는 칸 수
    # 사용할 수 없는 칸 수 = w + h - gcd(w, h)
    return w * h - (w + h - math.gcd(w, h))
```

### 코드 설명
- `math.gcd(w, h)` 함수를 사용하여 최대공약수를 쉽게 구합니다.
- 전체 격자 개수 `w * h`에서 대각선이 훼손한 격자 개수를 뺍니다.
- `int` 범위: Python은 큰 정수를 자동 처리하므로 $W, H$가 1억이어도 곱셈 결과 처리에 문제가 없습니다.
