# 문자열을 정수로 바꾸기

## 문제 설명
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12925)

문자열 `s`를 정수로 변환하세요. 부호(+, -)가 포함될 수 있습니다.

## 해결 전략
파이썬의 `int()`는 부호가 포함된 문자열도 자동으로 처리해 줍니다.

### 알고리즘 순서
1. return `int(s)`

## Python 코드

```python
def solution(s):
    return int(s)
```
