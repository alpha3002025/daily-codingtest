# 수박수박수박수박수박수?

## 문제 설명
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12922)

길이 `n`에 맞춰 "수박수박..." 패턴을 반환하세요.
- n=3 -> "수박수"
- n=4 -> "수박수박"

## 해결 전략
"수박" 문자열을 `n // 2 + 1`번 반복한 뒤, 앞에서 `n`글자만 자릅니다.
또는 홀/짝 인덱스로 붙입니다.

### 알고리즘 순서
1. `pattern` = "수박" * `(n // 2 + 1)`
2. return `pattern[:n]`

## Python 코드

```python
def solution(n):
    # 넉넉하게 만든 뒤 자르기
    return ("수박" * ((n // 2) + 1))[:n]
```
