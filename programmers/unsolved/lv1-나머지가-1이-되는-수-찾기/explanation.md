# 나머지가 1이 되는 수 찾기

## 문제 설명
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/87389)

자연수 `n`을 `x`로 나눈 나머지가 1이 되도록 하는 **가장 작은 자연수 `x`**를 반환하세요.
(3 ≤ `n` ≤ 1,000,000)

## 해결 전략
`n % x == 1`을 만족하는 가장 작은 `x`를 찾아야 합니다.
`x`는 적어도 2 이상이어야 합니다(나머지가 1이므로).
단순 반복문으로 `x`를 2부터 `n`까지 증가시키며 확인하면 됩니다. 가장 작은 값을 찾아야 하므로 찾자마자 반환(`return`)하면 종료됩니다.

### 알고리즘 순서
1. `x` from 2 to `n-1`:
    - `if n % x == 1`: return `x`

## Python 코드

```python
def solution(n):
    for x in range(2, n):
        if n % x == 1:
            return x
    return n # 도달하지 않음 (문제 조건상 항상 존재)
```

## 배운 점 / 팁
- **단순 구현**: 가장 직관적인 방법이 정답일 때가 많습니다. n이 백만 정도면 O(N) 탐색은 충분히 빠릅니다.
