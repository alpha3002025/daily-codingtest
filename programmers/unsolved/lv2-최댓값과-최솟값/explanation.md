# 최댓값과 최솟값

## 문제 설명
공백으로 구분된 숫자가 담긴 문자열 `s`가 주어집니다.
이 문자열에서 가장 작은 수와 가장 큰 수를 찾아 `"(최솟값) (최댓값)"` 형태의 문자열을 반환해야 합니다.

예: `"1 2 3 4"` -> `"1 4"`, `"-1 -2 -3 -4"` -> `"-4 -1"`

## 풀이 개념
문자열 파싱과 기본 수학 함수를 사용합니다.

1. **문자열 분리**: `split()` 함수로 공백 기준 문자열을 나눕니다.
2. **정수 변환**: 분리된 문자열 리스트를 `int`형으로 변환합니다 (`map` 사용).
3. **최소/최대 찾기**: `min()`, `max()` 함수를 이용해 가장 작은 수와 큰 수를 찾습니다.
4. **포맷팅**: 찾은 값을 다시 문자열로 변환하여 공백으로 이어 붙입니다.

## 코드 (Python)

```python
def solution(s):
    # 1. 공백 기준으로 나눈 뒤, 각 원소를 정수로 변환하여 리스트 생성
    arr = list(map(int, s.split()))
    
    # 2. 최솟값과 최댓값 찾기
    min_val = min(arr)
    max_val = max(arr)
    
    # 3. f-string을 사용하여 포맷팅 후 반환
    return f"{min_val} {max_val}"
```
