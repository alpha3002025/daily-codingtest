# 두 원 사이의 정수 쌍

## 문제 설명
반지름이 `r1`, `r2`인 두 원이 (0, 0)을 중심으로 놓여 있습니다 (`r1 < r2`).
두 원 사이의 공간(두 원의 경계 포함)에 존재하는 정수 좌표 `(x, y)`의 개수를 구하는 문제입니다.
즉, `r1^2 <= x^2 + y^2 <= r2^2` 를 만족하는 정수 쌍의 개수를 반환해야 합니다.

## 풀이 개념
**수학/기하학** 문제입니다. 1사분면(x > 0, y >= 0)의 점의 개수를 구한 뒤, 대칭성을 이용하여 전체 개수를 계산합니다.

1. **1사분면만 고려**: x좌표를 1부터 `r2`까지 순회합니다.
2. 각 x에 대해 가능한 y의 범위를 구합니다.
   - y의 최댓값 (`max_y`): 큰 원의 방정식 `x^2 + y^2 <= r2^2` 에서 `y <= sqrt(r2^2 - x^2)`. 정수여야 하므로 내림(`floor`) 처리합니다.
   - y의 최솟값 (`min_y`): 작은 원의 방정식 `x^2 + y^2 >= r1^2` 에서 `y >= sqrt(r1^2 - x^2)`.
     - 만약 `x < r1`이라면, `y >= sqrt(r1^2 - x^2)` 이므로 올림(`ceil`) 처리합니다.
     - 만약 `x >= r1`이라면, 작은 원 바깥쪽 x좌표이므로 y는 0부터 가능합니다. (`min_y = 0`)
3. 각 x에서의 유효한 y의 개수는 `max_y - min_y + 1` 입니다.
4. 이 값을 모두 더한 뒤 4를 곱하면 전체 정수 쌍의 개수가 됩니다.
   - 이 방식은 x축 위의 점(y=0, x>0)은 포함하지만 y축 위의 점(x=0)은 포함하지 않고 순회한 것입니다.
   - 4를 곱하면 4개의 사분면 각각에서 (해당 사분면 내부 + 한쪽 축)을 더하게 되어, 결과적으로 중복 없이 모든 점을 정확히 덮게 됩니다.

## 코드 (Python)

```python
import math

def solution(r1, r2):
    count = 0
    
    for x in range(1, r2 + 1):
        # 큰 원에 대한 y의 최대값 (내림)
        max_y = math.floor(math.sqrt(r2**2 - x**2))
        
        # 작은 원에 대한 y의 최소값 (올림)
        # x가 r1보다 크거나 같으면 작은 원의 영향 없이 0부터 시작 가능
        if x < r1:
            min_y = math.ceil(math.sqrt(r1**2 - x**2))
        else:
            min_y = 0
            
        count += (max_y - min_y + 1)
        
    return count * 4
```
