문제는 막상 풀어보면 쉬운데, 문제 읽을때 '얘 뭐라는거야?' 하면서 긴장하게 되는 문제. 이상하게 암호처럼 만들어놓은 문제였다.

# python
```python
def winner(c):
    if c == '2': # 가위
        return '0' ## 0 == 바위
    
    if c == '0': # 바위
        return '5' ## 5 == 보
    
    if c == '5': # 보
        return '2' ## 2 == 가위


def solution(rsp):
    return ''.join([winner(c) for c in rsp])
```
