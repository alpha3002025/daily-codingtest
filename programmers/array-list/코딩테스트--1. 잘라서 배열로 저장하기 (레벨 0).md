# python (책(1))
```python
def solution(my_str, n):
    answer = []
    index = 0
    while index < len(my_str):
        answer.append(my_str[index:index+n])
        index = index + n
    
    return answer
```

# python (책(2)) : 리스트 컴프리헨션

```python
def solution(my_str, n):
    return [my_str[i:i+n] for i in range(0, len(my_str), n)]
```

일단 풀긴 풀었는데, 다른 풀이도 보고 좀 배우자. 다른 사람의 풀이에 좋은게 많다.
```python
def solution(my_str, n):
    answer = []
    
    for i in range(len(my_str)//n + 1):
        curr = i*n
        s = ""
        if curr + n< len(my_str):
            s = my_str[curr:curr+n]
        else:
            s = my_str[curr:]
        
        if s != "":
            answer.append(s)
    
    return answer
```
