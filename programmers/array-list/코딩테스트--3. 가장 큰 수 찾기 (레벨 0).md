책의 풀이는 3가지
- (1) 가장 큰 수를 찾은 후 결과제출시 그 수의 인덱스를 찾는 방식 (가장 느림)
- (2) 내가 푼 방식과 같은 방식 (카운트)
- (3) enumerate, lambda 정렬을 이용한 풀이
	- 문법을 다양하게 알아야 해서 (3) 역시 필요하다. enumerate 는 i, num in enumerate(array) 와 같이 동작한다.
	- key = lambda ... 에 익숙해지자.

입력 제한 값이 1000 으로 되어 있기 때문에 (1),(3) 정도도 가능하다.

# python (my)
```python
def solution(array):
    max_number = 0
    max_index = 0
    for i in range(len(array)):
        if max_number < array[i]:
            max_number = array[i]
            max_index = i
    
    return [max_number, max_index]
```


# python (책 (3))
```python
def solution(array):
    [i, num] = list(sorted(enumerate(array), key= lambda x: x[1]))[-1]
    return [num, i]
```
