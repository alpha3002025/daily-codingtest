책은 두가지 방식으로 풀었다.
- (1) : 문자열로 반환 후 "7"의 개수 카운팅
- (2) : (1) 을 리스트 컴프리헨션으로

나는 나눗셈/나머지 개념을 사용했다.

문자열 방식도 알고 있어야 나중에 응용이 될 만한 부분이 많기 때문에 책의 풀이 2가지 풀이를 모두 풀어보았다.

# python
```python
def solution(array):
    answer = 0
    
    for n in array:
        while n > 0:
            if n % 10 == 7:
                answer+=1
            n //= 10
    
    return answer
```


# python (책 (1))
```python
def solution(array):
    answer = 0
    
    for n in array:
        s = str(n)
        for c in s:
            if c == "7":
                answer+=1
    
    return answer
```


# python (책 (2))
```python
def solution(array):
    return sum(str(num).count("7") for num in array)
```
