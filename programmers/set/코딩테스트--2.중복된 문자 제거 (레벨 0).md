# python (my)
너무 어렵게 생각하고 풀었다. 풀이도 좀 꼬이긴 했다.
```python
from collections import Counter

def solution(my_string):
    counter = Counter(my_string)
    
    answer = ''
    for c in my_string:
        if counter[c] > 1:
            counter[c] = 0
            answer += c
        elif counter[c] == 1:
            answer += c
            counter[c] = 0
    return answer
```


# python (책 (1))
```python
def solution(my_string):
    occurence = set()
    answer = ''
    
    for s in my_string:
        if s in occurence:
            continue
            
        occurence.add(s)
        answer += s
        
    return answer
```


# python (책(2))
dictionary 를 이용한 방식인데 이 방식은 문법을 기억해두면 언제든지 쓸모가 많아보인다. 
```python
def solution(my_string):
    return ''.join(dict.fromkeys(my_string))
```
