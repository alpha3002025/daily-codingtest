# 8. 문자열 나누기 (이해는 함. 계속 연습)
알고보니 답봤던 문제였다. 이번 풀이 때는 어느 정도 이해가 갔다.
for loop 을 잘쓴다고 착각했던 나를 반성한다. 그냥 문법을 알고 있던 거였음 하하하...

문제를 풀기 위해 초기 세팅 이후로 초기값이 지나가게 만든 부분이 인상적이었다.
```python
def solution(s):
    count = 0
    me = 0
    other = 0
    
    for curr in s:
        if me == other: ## (1)
            count += 1
            pre = curr ## 다른 요소의 개수가 같아지면, i-1 까지의 결과가 x:x-other 카운트가 같다는 의미
        
        ## (2)
        if curr == pre: ## 첫번째 요소는 여기를 들른다. 이후부터는 else, 그 이후로는 문자열에 따라 다르다.
            me += 1
        else:
            ## ...
    
    return count 
```

(1)
- 인덱스가 0부터 시작한다고 가정하고 처음 카운트는 me, other = 0 으로 시작하는데, 여기에 대해서 `me==other` 이기에 카운트가 된다. 그리고 pre 역시 변경이 되는데 따라서 (2) 로 이동한다.
(2)
- 현재 문자가 이전값과 같을 경우 (`curr == pre`) me 를 + 한다. 의미는 me 가 나타난 횟수를 카운트한다는 의미다.

```python
def solution(s):
    count = 0
    me = 0
    other = 0
    
    for char in s:
        if me == other:
            count += 1
            pre = char
        
        if char == pre:
            me += 1
        else:
            other += 1
            
    return count
```
