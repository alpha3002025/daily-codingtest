# python - 풀이 1
- split() 대신 `replace(" ", "")` 을 쓸 경우 오답 처리 되는 반례를 찾아야 한다!!

```python
def solution(my_string):
    answer = 0
    query = my_string.split()
    op = ""
    
    for q in query:
        if q in ['+', '-']:
            op = q
        else:
            if op == "+":
                answer += int(q)
            elif op == '-':
                answer -= int(q)
            else:
                answer = int(q)
                
    return answer
```

# python - 풀이 2 (참고 : 런타임 에러)
```python
def solution(my_string):
    query = my_string.replace("-", "+ -").split("+")
    
    answer = 0
    
    for q in query:
        answer += int(q)
    
    return answer
```


# kotlin
```kotlin
class Solution {
    fun solution(my_string: String): Int {
        return my_string
            .replace("- ".toRegex(), "-")
            .replace("[+] ".toRegex(), "")
            .split(" ")
            .sumOf{it.toInt()}
    }
}
```
