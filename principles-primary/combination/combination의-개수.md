# 조합(Combination)의 개수
**조합(Combination) 공식** : $_nC_2 = \frac{n(n-1)}{2}$
- 같은 몸무게를 가진 사람이 `v`명 있을 때, 이 중 2명을 뽑아 짝을 짓는 경우의 수입니다.
- 예: 100kg인 사람이 `A`, `B`, `C` 3명(`v=3`) 있다면?
  - (A, B), (A, C), (B, C) -> 총 3쌍.
  - 계산: $3 \times (3-1) / 2 = 3$.

## e.g. lv2. 시소 짝꿍 - 조합공식 말고 다른 공식으로 착각했었다
내 경우에는 같은 몸무게 100kg 인 사람 3 명을  2명을 뽑아서 시소에 태우는 경우를 생각해봤다. 아!! 물론 시소는 쇠로 되어 있어서 부러지진 않는다고 가정했다. 나는 아래 공식을 봤을때 단순히 수열의 합을 구하는 공식인 줄 착각했다. 나이 먹어서 그래... 나이먹어봐... 공식이 막 다른 공식으로 보여...
```python
for k, v in counter.items():
    if v > 1:
        answer += v * (v - 1) // 2
```

그리고 막상 생각해보면 (v x (v+1))/2 가 원래 내가 아는 수열의 합 공식인데, 내 기억?추억?에 어떻게든 끼워맞춰서 이해하려고 했던 나의 지난 10분간...은 시간낭비였다. Gemini 한테 물어보고 난 후에야 '아!! 이게 조합 공식이네? 어? 비슷하게 생겼네?' 라는 생각이 들었다. 나이 먹어서 그려... 나이 먹으면 디져야지...<br/>
<br/>

그런데 100kg 인 사람 3 명 중 2명을 뽑아서 시소에 태우는 경우는 다음과 같다.
  - (A, B), (A, C), (B, C) -> 총 3쌍.

그리고 이건 공식으로는 v * (v-1) / 2 로 표현할수 있다.
- $v \times (v-1) / 2 = 3$
